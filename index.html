Key Features

Private & Secure

Your data stays on your device. No third-party servers involved.

Flexible Operation

Choose between HTTP and MQTT modes to best suit your needs.

Simple Integration

RESTful API and MQTT protocol support for easy integration.

Open Source

Free and open-source software with active community support.

Operating Modes

HTTP Mode

Classic REST API mode where your phone acts as an HTTP server. Perfect for direct integration and local network usage.

  • Direct HTTP endpoints
  • Simple REST API
  • Ideal for local networks

MQTT Mode

Connect to an MQTT broker for reliable message delivery and real-time communication. Great for Internet-connected scenarios.

  • Broker-based messaging
  • Real-time communication
  • Perfect for Internet connectivity

Command Line Interface

Easy command-line access to SMS Gateway

Install the CLI tool using npm:

npm install -g @capcom6/sms-gateway-cli

Example Usage

# Send an SMS message sms-gateway send --to "+1234567890" --text "Hello from CLI!"
# Configure the client sms-gateway config set --url "http://192.168.1.100:8080" --token "your-auth-token"

Quick Start

curl -X POST http://your-phone-ip:8080/api/v1/messages \
-H "Content-Type: application/json" \
-H "Authorization: Bearer your-token" \
-d '{
  "to": "+1234567890",
  "text": "Hello from SMS Gateway!"
}'
